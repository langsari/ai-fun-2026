\# AI Mood Detection System



\## Student Information

Name: Bangaly Sanoh



\## Project Overview

This project is an AI-based system that detects human emotions using a webcam. 

It uses computer vision and machine learning to analyze facial expressions in real-time.



\## Objectives

\- Detect human face using a camera

\- Analyze emotions using AI

\- Display results live on screen



\## Technologies Used

\- Python

\- OpenCV

\- FER (Facial Emotion Recognition)

\- NumPy



\## Project Structure

\- data/: stores datasets

\- notebooks/: experimentation and analysis

\- src/: main source code

\- reports/: results and screenshots



\## Methodology

1\. Capture video using webcam

2\. Detect faces in each frame

3\. Use FER model to predict emotions

4\. Display the dominant emotion



\## Results

The system successfully detects:

\- Happy

\- Sad

\- Angry

\- Surprise

\- Neutral



\## Future Improvements

\- Improve accuracy with custom dataset

\- Add GUI interface

\- Deploy as web application



\## Conclusion

This project demonstrates how AI can be used for real-time emotion detection and human-computer interaction.
