# 🕌 Quran Memorization Performance Prediction

## 📌 Overview

This project uses Machine Learning to predict whether a student will have good or weak Quran memorization performance based on daily habits.

---

## 🎯 Objective

To build a classification model that predicts memorization performance using:

* Study hours
* Revision hours
* Sleep hours
* Consistency

---

## 📂 Dataset

The dataset contains:

* `study_hours`
* `revision_hours`
* `sleep_hours`
* `consistency`
* `memorization` (target)

---

## ⚙️ Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

---

## 🧠 Machine Learning Model

* Logistic Regression
* Train/Test Split (80/20)

---

## 📊 Features

* Data visualization (histograms)
* Model training
* Accuracy evaluation
* User input prediction

---

## 🚀 How to Run

1. Install dependencies:

```
pip install pandas matplotlib scikit-learn
```

2. Run the script in Jupyter Notebook

3. Enter input values:

* Study hours
* Revision hours
* Sleep hours
* Consistency

---

## 📈 Example Output

```
Prediction: Good Memorization
```
