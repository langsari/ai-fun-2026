Project Title: Gold Price Prediction Using Machine Learning
1. Introduction

This project focuses on predicting gold prices using machine learning techniques. Gold price prediction is important in finance and investment, as it helps investors make better decisions based on data analysis and market trends.

2. Objective

The main objective of this project is to build a machine learning model that can predict future gold prices based on historical data and important economic factors. The project also aims to compare different models and evaluate their performance.

3. Methodology

In this project, the following steps were applied:

Data Collection: Historical data of gold prices and other related economic factors were collected.
Data Preprocessing: The data was cleaned and prepared for model training.

Feature Selection: The prediction is based not only on gold prices but also on important influencing factors such as:

Oil prices
US dollar value
Other economic indicators

These features help the model understand the relationship between gold and global economic conditions.

Model Training: Several machine learning models were used, including:
Random Forest
Decision Tree
K-Nearest Neighbors (KNN)
Gradient Boosting
Model Evaluation: The models were evaluated using performance metrics such as:
Mean Absolute Error (MAE)
Root Mean Squared Error (RMSE)
4. Results

The models were trained and tested, and their performance was compared. The results showed that some models performed better than others in predicting gold prices. The use of multiple features (such as oil and dollar values) improved the prediction accuracy.

5. Conclusion

This project demonstrates how machine learning can be used to predict gold prices by analyzing both historical data and external influencing factors. Although the predictions are not 100% accurate, the models provide useful insights for financial decision-making.

6. Future Improvements

This project can be improved in the future by:

Using more data for better accuracy
Adding more influencing factors (e.g., inflation, interest rates)
Using advanced models such as deep learning
Improving model tuning and optimization

From this project, I learned:

How to use machine learning models
