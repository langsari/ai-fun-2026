# AI Workout Readiness Analyzer

## Project Idea
This project uses AI to analyze whether a user should:
- Rest
- Do light workout
- Do intense workout

based on their physical condition.

## Input Data
- Sleep hours
- Energy level
- Muscle soreness
- Stress level
- Consecutive workout days
- Weight, height, BMI

## Process
The system uses:
1. AI model (Gemini API)
2. Fallback scoring system

## Output
- Workout recommendation
- Explanation
- Suggested activity

## Example Result
Decision: Light Workout  
Reason: Low energy and high soreness  
Recommendation: Stretching and light cardio
